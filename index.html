üî§ Basic Syntax

Hello World

Every programming journey starts with "Hello World":

say "Hello, World!"
Hello, World!

Comments

Comments help explain your code. They are ignored by the interpreter:

# This is a comment say "This will be printed" # say "This won't be printed"
Comments start with # and continue to the end of the line. Use them to document your code!

üì¶ Variables

Variables store data that can be used later in your program. In DoroLang, you declare variables using the kas keyword.

Declaring Variables

kas name = "Alice" kas age = 25 kas is_student = true say "Name: " + name say "Age: " + age say "Student: " + is_student
Name: Alice Age: 25 Student: true

Data Types

DoroLang supports three main data types:

Type Example Description
Numbers 42, 3.14 Integers and floating-point numbers
Strings "hello", 'world' Text enclosed in single or double quotes
Booleans true, false Logical true/false values

Example: Working with Variables

kas first_name = "John" kas last_name = "Doe" kas full_name = first_name + " " + last_name say full_name
John Doe

üîÄ Conditional Statements

Conditional statements allow your program to make decisions based on conditions.

If Statement

The basic conditional structure:

if (condition) { # code to execute if condition is true }

Example: Simple If Statement

kas age = 20 if (age >= 18) { say "You are an adult" }
You are an adult

If-Else Statement

Execute different code based on condition:

if (condition) { # code if true } else { # code if false }

Example: If-Else

kas score = 85 if (score >= 60) { say "You passed!" } else { say "You failed" }
You passed!

Nested Conditionals

You can nest if statements inside other if statements:

kas age = 25 kas has_license = true if (age >= 18) { if (has_license) { say "You can drive!" } else { say "You need a license" } } else { say "You are too young" }
Use parentheses around conditions for clarity, though they're optional in DoroLang.

üîÑ Loops

Loops allow you to repeat code multiple times. DoroLang supports while and for loops.

While Loop

Repeats code while a condition is true:

while (condition) { # code to repeat }

Example: While Loop

kas counter = 1 while (counter <= 5) { say "Count: " + counter kas counter = counter + 1 }
Count: 1 Count: 2 Count: 3 Count: 4 Count: 5
Be careful with while loops! Make sure the condition eventually becomes false, or you'll create an infinite loop.

For Loop

Repeats code a specific number of times:

for kas variable = start to end { # code to repeat }

Example: For Loop

say "Counting from 1 to 5:" for kas i = 1 to 5 { say "Number: " + i }
Counting from 1 to 5: Number: 1 Number: 2 Number: 3 Number: 4 Number: 5

For Loop with Step

You can specify a step value to count by increments:

for kas i = 0 to 10 step 2 { say i }
0 2 4 6 8 10

Nested Loops

Loops can be nested inside other loops:

Example: Multiplication Table

for kas i = 1 to 3 { for kas j = 1 to 3 { kas result = i * j say i + " x " + j + " = " + result } say "---" }
1 x 1 = 1 1 x 2 = 2 1 x 3 = 3 --- 2 x 1 = 2 2 x 2 = 4 2 x 3 = 6 --- 3 x 1 = 3 3 x 2 = 6 3 x 3 = 9 ---

üîß Functions

Functions allow you to organize code into reusable blocks. You can define functions and call them multiple times with different arguments.

Defining Functions

Use the function keyword to define a function:

function function_name(parameter1, parameter2) { # function body return value }

Example: Simple Function

function greet(name) { return "Hello, " + name + "!" } say greet("Alice") say greet("Bob")
Hello, Alice! Hello, Bob!

Functions with Multiple Parameters

Example: Calculator Functions

function add(a, b) { return a + b } function multiply(x, y) { return x * y } say "Sum: " + add(5, 3) say "Product: " + multiply(4, 7)
Sum: 8 Product: 28

Return Statement

The return statement exits the function and returns a value:

function max(a, b) { if (a > b) { return a } else { return b } } say max(10, 5)
10

Functions Without Return Value

Functions can also be used to perform actions without returning a value:

Example: Function for Display

function print_header() { say "==========" say " HEADER" say "==========" } print_header() say "Content here" print_header()
========== HEADER ========== Content here ========== HEADER ==========

Complex Functions

Example: Function with Conditionals

function check_age(age) { if (age >= 18) { return "Adult" } else { if (age >= 13) { return "Teenager" } else { return "Child" } } } say check_age(25) say check_age(15) say check_age(8)
Adult Teenager Child

Functions with Loops

Example: Function Using Loops

function count_to_n(n) { for kas i = 1 to n { say i } } count_to_n(5)
1 2 3 4 5
Functions help organize your code and make it reusable. Define functions for tasks you need to perform multiple times!

‚å®Ô∏è User Input

The input() function allows your program to get data from the user.

Basic Input

kas name = input("What is your name? ") say "Hello, " + name + "!"

Example: Interactive Program

kas user_name = input("Enter your name: ") kas age_str = input("Enter your age: ") kas age = age_str say "Welcome, " + user_name + "!" if (age >= 18) { say "You are an adult" } else { say "You are a minor" }
The input() function always returns a string. If you need a number, DoroLang will automatically convert it when used in arithmetic operations.

You guessed it!" } else { if (guess_num < secret_number) { say "Too low! Try again." } else { say "Too high! Try again." } }

Example 3: Countdown Timer

Example 4: Grade Calculator

# Calculate and display grade kas score = 85 kas attendance = 90 say "Score: " + score say "Attendance: " + attendance + "%" kas excellent = score >= 90 and attendance >= 85 kas good = (score >= 80 and attendance >= 80) or score >= 85 if (excellent) { say "Grade: A - Excellent work!" } else { if (good) { say "Grade: B - Good job!" } else { say "Grade: C - Needs improvement" } }

Example 5: Factorial Calculator

# Calculate factorial of a number kas n = 5 kas factorial = 1 kas i = 1 while (i <= n) { kas factorial = factorial * i kas i = i + 1 } say "Factorial of " + n + " is " + factorial
Factorial of 5 is 120

Example 6: Function-Based Calculator

# Calculator using functions function add(a, b) { return a + b } function subtract(a, b) { return a - b } function calculate(operation, x, y) { if (operation == "add") { return add(x, y) } else { return subtract(x, y) } } say "10 + 5 = " + calculate("add", 10, 5) say "10 - 5 = " + calculate("subtract", 10, 5)
10 + 5 = 15 10 - 5 = 5

Example 7: Recursive Function

# Calculate factorial using function function factorial(n) { if (n <= 1) { return 1 } else { return n * factorial(n - 1) } } say "Factorial of 5: " + factorial(5)
Factorial of 5: 120

üéì Learning Path

Follow this learning path to master DoroLang:

  1. Start with basics: Learn how to use say and variables
  2. Master operators: Understand arithmetic, comparison, and logical operators
  3. Control flow: Learn if-else statements
  4. Loops: Practice with while and for loops
  5. Functions: Learn to define and use functions
  6. Input: Make your programs interactive
  7. Combine everything: Build complex programs using all concepts
Practice is key! Try modifying the examples, break them, fix them, and create your own programs. Start simple and gradually build more complex programs.
